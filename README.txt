# Music Database Application

A full-stack web application that allows users to search and view songs and albums, with features like a "song of the day" and advanced search options for songs based on various filters like duration, plays, danceability, energy, and valence. The project utilizes React for the frontend and PostgreSQL for the backend.

## Features

- **Song of the Day**: Displays a randomly selected song each day.
- **Search Songs**: Users can filter songs by title, duration, plays, danceability, energy, valence, and explicit content.
- **Top Songs and Albums**: Displays a list of top songs and albums based on number of plays.
- **Lazy Loading**: Implements lazy loading for data to optimize performance.
- **Responsive UI**: Built with Material-UI and Recharts for charts and responsive tables.
- **Album Info Page**: View album details and all songs in the album.

## Technologies Used

- **Frontend**: React, Material-UI
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL
- **Charts**: Recharts (for bar and radar charts)
